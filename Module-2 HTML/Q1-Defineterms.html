Q1:Define the terms:


Website: Website is a collection of related web pages that may contain tex,image,audio and video.A website can consist of one page,or thousands of pages,depending on what the site owner is trying to accomplish.

Webpage: Webpages is a digital document that contains information,usually written in HTML,and can be viewed by anyone connected to the internet through a web browser.

Web browser: Web browser is a computer program that helps the user to find information from the internet.

Web server: Web server is a softwer that provides web pages and websites to users when a users when a user ask and helps people surf the net.

HTML: The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript.

CSS: Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.
